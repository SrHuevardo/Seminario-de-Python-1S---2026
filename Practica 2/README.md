# Seminario de Python CDO - UNLP - Práctica 2

## Instalación y Configuración

Para ejecutar este proyecto, se recomienda el uso de un entorno virtual para mantener las dependencias aisladas.

### 1. Crear y activar el entorno virtual .venv

#### Windows
```PowerShell
python -m venv .venv
.\.venv\Scripts\activate
```
#### Linux 
```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 2. Instalar dependencias

Una vez activado el entorno, instalar los paquetes necesarios.

```Bash
pip install -r requirements.txt
```

### 3. Ejecutar lo programas

Para visualizar y correr los ejercicios, se puede utilizar Visual Studio Code con la extensión de Jupyter o iniciar el servidor de Jupyter Lab

```Bash
jupyter lab
```
